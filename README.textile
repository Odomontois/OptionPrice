h1. Расчёт стоимости опционов

Разработка алгоритма расчёта стоимости опционов с изменением цены, зависящим от времени

h2. Результат для модифицированных данных ( с нулевыми узлами )

* Оптимизированный алгоритм:

code. (125.77041123686298,562)

* Неоптимизированный алгоритм( контрольный )

code. (125.77041123686314,797161)

Первое число показывает результат, второе - количество обработанных узлов.
Очевидно, что оптимизированный алгоритм охрененно оптимизирует там всё, не ломая результат.
